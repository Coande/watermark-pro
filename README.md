## ✨ Feather

- [x] Grid System
- [x] WaterMark
- [x] Mosaic
- [x] WaterMark Whole Page
- [x] Multi File
- [x] No Backend & Pure Frontend (Super Security)
- [ ] Support watermark with image
- [ ] Mobile / Weixin APP
- [ ] Electron Desktop App

## 🔨 Contribution

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```
